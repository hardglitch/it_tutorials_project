The project is written in Python and is completely asynchronous.
For the front, Jinja2 and pure JavaScript were used.

Stack
- FastAPI
- Uvicorn
- SQLAlchemy + asyncpg
- Redis (Hiredis)
- PostgreSQL
- Jinja2
- HTML, CSS, JavaScript
